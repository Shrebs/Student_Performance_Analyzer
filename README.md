## 🧠 Overview

The **Student Performance Analyzer** is a data analytics project that evaluates student performance using key factors such as study hours, sleep duration, and attendance.
It compares previous and current exam scores to uncover performance trends and delivers insights through visualizations and an interactive dashboard.

---

## 🎯 Objectives

* 📌 Compare previous and current exam performance
* 📉 Identify trends of improvement or decline
* 📊 Analyze factors affecting performance
* 🎯 Provide both overall and student-level insights
* 🧩 Build an interactive dashboard for exploration

---

## ✨ Features

* 📈 Scatter plots for performance analysis
* 📊 Histogram for score difference distribution
* 🔥 Heatmap for correlation analysis
* 📉 Attendance-based performance visualization
* 🎯 Individual student performance tracker
* 🧩 Interactive dashboard with student selection

## 📂 Dataset Description

The dataset contains **200 student records** with the following attributes:

| Column Name        | Description               |
| ------------------ | ------------------------- |
| student_id         | Unique student identifier |
| hours_studied      | Number of hours studied   |
| sleep_hours        | Average sleep duration    |
| attendance_percent | Attendance percentage     |
| previous_scores    | Previous exam score       |
| exam_score         | Current exam score        |

## ⚙️ Installation & Usage

### 🔹 Clone the Repository

```bash
git clone git@github.com:Shrebs/Student_Performance_Analyzer.git
cd Student_Performance_Analyzer
```

### 🔹 Install Dependencies

```bash
pip install pandas matplotlib seaborn streamlit
```

### 🔹 Run the Dashboard

```bash
streamlit run Dashboard.py
```

## 📈 Key Insights

* 📉 All students showed a **decline in performance** in the final exam
* 📊 The average score dropped significantly across all records
* ⚠️ The consistent decline suggests **higher exam difficulty or external factors**
* 📌 Performance trends remain uniform despite variations in study habits

---

## 🎯 Individual Student Analysis

The dashboard allows users to:

* Select a **student_id**
* View **previous vs current score comparison**
* Analyze **score difference**
* Get a **performance conclusion (Improved / Declined / No Change)**
